# Admin App

Admin and instructor React application for Presstronic Academy.

## Scope

`apps/admin` owns content management, content health, support, reporting, moderation, and administrative route surfaces. Shared reusable primitives come from `@presstronic-academy/ui`; backend clients, auth integration, persistence, and production admin workflows remain deferred until focused proposals accept them.

## Commands

- `pnpm --filter @presstronic-academy/admin dev` starts Vite on port `5174`.
- `pnpm --filter @presstronic-academy/admin build`
- `pnpm --filter @presstronic-academy/admin lint`
- `pnpm --filter @presstronic-academy/admin typecheck`
- `pnpm --filter @presstronic-academy/admin check`
