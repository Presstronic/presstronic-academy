# UI Package

Shared ShadCN-based React component package for `apps/web` and `apps/admin`.

## Scope

`packages/ui` owns reusable Academy primitives, styling utilities, visual tokens, and source-owned ShadCN-compatible component wrappers. App routes, workflow state, data loading, and feature-specific compositions stay in the consuming app.

## Commands

- `pnpm --filter @presstronic-academy/ui build`
- `pnpm --filter @presstronic-academy/ui lint`
- `pnpm --filter @presstronic-academy/ui typecheck`
- `pnpm --filter @presstronic-academy/ui check`
