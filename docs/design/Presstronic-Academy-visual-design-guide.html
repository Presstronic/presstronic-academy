00 / Foundations

Foundation

Night City at a desk job

Operational, dense, sharp, and legible. The product should feel like a learning instrument panel, not a course marketplace, brochure, or generic SaaS dashboard.

Design law

Square geometry

Radius is 0 by default. Use the notched top-right corner only for marquee panels and irreversible moments.

Cyan does work

Cyan owns primary actions, keyboard focus, active navigation, progress, and selected controls.

Mono means system

IBM Plex Mono is for labels, metadata, IDs, telemetry, code, state names, and compact controls.

No empty theatre

Use compact panels, rows, grids, dividers, and anchored controls. Avoid giant blank space and hero-card staging.

Themes

Dark and light are both specified

Dark is the default. Light is a full operational mapping, not an incomplete inversion.

Spacing, shape, and motion

Compact, square, predictable

Everything should feel engineered: clear rows, stable dimensions, visible focus, and short motion.

8px grid

Use 4, 8, 12, 16, 24, 32, 48, 64. Page padding is clamp(28px, 3vw, 48px).

Radius 0

No rounded cards, pills, or soft SaaS capsules. The system is square by default.

Notch

One clipped top-right corner for marquee panels, dangerous confirmation, or narrative gates.

Motion

120ms hover/focus, 180ms overlays, 220ms page transitions. Respect reduced motion.
