Academy's action button — square, mono uppercase label; use \`notch\` on the single most important CTA of a screen.

```jsx
<Button variant="primary" size="lg" notch>Begin mission</Button>
<Button variant="ghost">View syllabus</Button>
```

Variants: primary (cyan fill), secondary (graphite), ghost (hairline border), destructive (red). Sizes sm/md/lg. \`glow\` adds a subtle cyan halo (primary only).
